## Creador y Propietario

### CodeIbra

**Backend Template** es un proyecto creado, diseñado y mantenido por **CodeIbra**, una iniciativa enfocada en el desarrollo de infraestructura backend moderna, escalable y preparada para entornos empresariales.

| Rol | Nombre/Alias |
|---|---|
| Creador Original | CodeIbra |
| Arquitecto Principal | CodeIbra |
| Propietario del Proyecto | CodeIbra |
| Email de Contacto | — |
| Sitio Web | — |
| Redes Sociales | — |

CodeIbra es el arquitecto detrás de la visión híbrida monolito + microservicios, el diseño del sistema multi-agente de IA, y la implementación del stack completo de observabilidad. El proyecto nació de la necesidad de contar con una plantilla backend que integrara de forma nativa diagnóstico automático de errores, recuperación autónoma, trazabilidad distribuida y analítica avanzada, todo bajo una misma base de código.

## Contribuciones Especiales

### Agentes de IA

El sistema **AI Error Doctor** fue posible gracias a la integración de modelos de lenguaje avanzados:

- **OpenAI GPT-4 / GPT-4o** — Análisis de errores y generación de parches.
- **Anthropic Claude 3.5 Sonnet** — Diagnóstico de causa raíz y sugerencias de fix.
- **OpenCode (Anthropic)** — Asistencia en el desarrollo del código base.

### Frameworks y Librerías Clave

- **NestJS** — Framework principal del monolito.
- **Express** — Framework HTTP para microservicios.
- **Prisma ORM** — Capa de acceso a datos.
- **OpenTelemetry** — Trazabilidad distribuida.
- **Prometheus + Grafana** — Métricas y visualización.
- **BullMQ** — Sistema de colas y jobs asíncronos.
- **Winston + Loki** — Logging estructurado.
