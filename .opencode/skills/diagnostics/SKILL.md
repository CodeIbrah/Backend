## What I do

- Analizar CPU, memoria, heap y event loop lag de servicios NestJS
- Identificar cuellos de botella en PostgreSQL, Redis y BullMQ
- Detectar memory leaks mediante tasa de crecimiento de heap
- Correlacionar métricas con logs para diagnóstico de causa raíz
- Generar reportes de salud con score por componente

## When to use me

Use this when investigating performance degradation, memory leaks, high latency, or system bottlenecks. Run on-demand or triggered by alerts from Runtime Monitoring.
