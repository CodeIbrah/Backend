## What I do

- Identificar slow queries mediante análisis de logs de Prisma
- Diagnosticar problemas de connection pool y lock contention
- Resolver errores de migraciones y esquemas
- Detectar N+1 queries y sugerir optimizaciones con eager loading
- Analizar deadlocks y transacciones bloqueadas

## When to use me

Use this when database queries are slow, migrations fail, connection pool exhausts, or Prisma throws unexpected errors. Also for optimizing query performance.
