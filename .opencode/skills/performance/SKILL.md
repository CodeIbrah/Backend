## What I do

- Profiling de latencia por endpoint con percentiles p50, p95, p99
- Optimizar consultas Prisma y estrategias de caché Redis
- Recomendar escalado horizontal y ajuste de recursos
- Detectar degradaciones mediante comparación con baseline
- Analizar patrones de garbage collection y event loop

## When to use me

Use this when endpoints show high latency, throughput drops, or resource usage is suboptimal. Provides optimization recommendations backed by profiling data.
