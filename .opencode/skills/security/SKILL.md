## What I do

- Sanitizar inputs y detectar inyecciones (SQL, XSS, NoSQL)
- Detectar secretos y credenciales expuestas en logs o código
- Analizar anomalías de autenticación (brute force, token reuse)
- Identificar requests sospechosos por patrón y geolocalización
- Prevenir abuso con rate limiting y bloqueo de IPs

## When to use me

Use this when investigating potential security incidents, detecting unusual auth patterns, sanitizing inputs, or reviewing the system for exposed secrets.
