## What I do

- Instrumentar métricas de aplicación con Prometheus y shared-telemetry
- Configurar logging estructurado con Winston y Loki
- Propagar correlation IDs y traces con OpenTelemetry y Jaeger
- Construir dashboards en Grafana para monitoreo visual
- Integrar los tres pilares (métricas, logs, traces) para debugging

## When to use me

Use this when setting up or troubleshooting observability: adding new metrics, configuring log shipping, tracing requests across services, or building Grafana dashboards.
